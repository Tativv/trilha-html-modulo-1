Introdução ao HTML

Não é uma linguagem de programação. É uma linguagem de marcação que informa aos navegadores como estruturar as páginas que você está visitando.

Como é uma estrutura básica de um HTML?


            <!doctype html> 
            <html>
            <head>
            <meta charset="utf-8" />
            <title>Mi página de prueba </title>
            </head>
            <body>
            </body>
            </html>
  1. <!doctype html>:

    Atualmente é ignorado e considerado um legado histórico que deve ser incluído para que tudo funcione corretamente. É a sequência de caracteres mais curta aceita como um elemento doctype válido. Isso é tudo que você realmente precisa saber.


  2. <html></html>: Este elemento envolve todo o conteúdo da página.

  3. <head></head>: É um contêiner para todos os parâmetros incluídos no documento HTML que não ficarão visíveis aos visitantes da página.

  4. <meta charset="utf-8" />: Está configurado para gerenciar qualquer conteúdo textual colocado no documento.

  5. <title></title>: Isso define o título da página, que é o título que aparece na aba do navegador em que a página é carregada.

  6. <body></body>: Ele contém todo o conteúdo que você deseja exibir aos usuários quando eles visitam sua página, seja texto, imagens, vídeos, jogos, faixas de áudio reproduzíveis ou qualquer outra coisa.

    Títulos e parágrafos

    Títulos

    Cada seção deve ser delimitada por um elemento de cabeçalho. Existem seis elementos de cabeçalho <h1>, <h2>, <h3>, <h4>, <h5> e <h6>.

    Págrafos

    Cada parágrafo deve ser delimitado por um elemento <p>.

    Entendendo a diferença entre listas ordenadas e não ordenadas

    <ol></ol>

    Permite definir listas ou marcadores ordenados ("Lista Ordenada"), numerados ou alfabeticamente.

    
    <ol>
    <li> Elemento 1</li> 
    <li> Elemento 2</li> 
    <li> Elemento 3</li> 
    </ol>
    <ul></ul>

    Da "lista não ordenada" - não ordenada. Cria uma lista não ordenada.

    
    <ul>
    <li> Elemento 1</li> 
    <li> Elemento 2</li> 
    <li> Elemento 3</li> 
    </ul>
